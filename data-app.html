Pearson/Python

Pearson/Python is an easy-to-use tool that lets you see trends in word use over time in the diary of Jonathan Pearson (1813-87), Union alumnus, professor, librarian, and treasurer.

I made this web application in 2021: the analysis and visualization are done via Python, the web interface is done via Streamlit, and the diary data is made available by Schaffer Library.

Pearson/Python allows you to search for one or more words in the diaries, and you can change settings, such as the date range, or whether to lemmatize or lower case the text or not.

Instructions are coming soon.
Please feel free to try out the tool and search the Pearson diaries!
